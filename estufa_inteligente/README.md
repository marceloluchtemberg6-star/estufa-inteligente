# 🚀 Estufa Inteligente - IoT

## 📌 Sobre o projeto
Sistema de automação de estufa com ESP32, sensores e MQTT.

---

## ⚙️ Funcionalidades
- Leitura de temperatura (DHT22)
- Leitura de luminosidade (LDR)
- Controle de LED automático
- Irrigação via MQTT
- Sistema de REARME remoto

---

## 📡 MQTT

Broker:
test.mosquitto.org

Tópicos:
- marcelo/estufa/temp
- marcelo/estufa/luz
- marcelo/estufa/cmd
- marcelo/estufa/rearme

---

## 🔁 Comandos MQTT

- IRRIGAR_ON
- IRRIGAR_OFF
- REARME
